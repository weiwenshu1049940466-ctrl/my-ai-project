# AI Water Detection Project

## 📌 프로젝트 설명
본 프로젝트는 Teachable Machine을 활용하여 이미지 인식 모델을 학습시키고, 수도꼭지의 상태(물 끔 / 정상적인 물 흐름 / 미세한 물 흐름)를 분류하는 시스템을 구현한 것이다.

## 🎯 활용 목적
본 프로젝트는 다음과 같은 분야에 활용될 수 있다:
- 절수 모니터링 시스템
- 스마트 홈 환경
- 공공장소의 수자원 관리

## 🧠 사용 기술
- Teachable Machine
- TensorFlow.js
- HTML / JavaScript
